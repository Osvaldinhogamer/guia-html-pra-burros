principais tags HTML

O html possui varias tags importantes, aqui estão algumas:

  • <h1>...</h1> titulos, do h1 ao h6
  • <p> - paragrafo de texto
  • <a href="image.jpg"> - links
  • <ul> <li> - listas não ordenadas
  • <ol> <li> - listas ordenadas
  • < table > - tabelas
  • <form> - formulario
  • <div> - containers genericos
  • <span> - container inline

Principais layouts em CSS

com CSS você pode organizar os elementos com diferentes formas:

  • Box Mode - toda tag e uma caixa com margin border, padding e content
  • Flexbox - permite criar layouts flexiveis em linha ou coluna.
  • CSS Grid - layout baseados em grandes bidimensionais.
  • Position - define a posição (static, relative, absolute, find, sticky)
  • float - metodo antigo de alinhar os elementos

exemplos de Flexbox

            .container {
              display: flex;
              justify-content: space-between;
              aling-items> center;
            }

exemplo grid

            .container {
              display: grid;
              grid-template-columns: 1fr 1fr 1fr;
              gap: 10px;
            }
